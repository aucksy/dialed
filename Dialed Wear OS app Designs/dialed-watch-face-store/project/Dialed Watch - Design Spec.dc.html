Dialed — Wear OS Companion · Design Spec · v1

A concierge on the wrist, not a store.

The receiving end of the Dialed phone app. Three jobs, done beautifully: permissions, the catch (receive → install → success), and activation. Round-first, true-black, one champagne-gold accent. Everything maps 1:1 to Wear Compose Material 3 (Expressive) — tokens and motion in the handoff and HANDOFF-WATCH.md.

Assumptions
  • Wear OS 6+ · Pixel Watch 4 reference · Wear Compose M3 Expressive
  • Mocks drawn at 2× on a 192dp round canvas (384px). Verify at 225dp XL
  • Watch holds one Dialed face at a time — no catalog here
  • Background is true black (#000) for AMOLED battery
  • Starring face is a placeholder — swap via Tweaks

C · Receiving a face

Build order: Theme/Tokens → FaceDial (shared) → HomeScreen → FirstRun → ReceiveFlow + W1 → Concierge (one-tap, then coaching loop) → Unsupported. Accessibility: contrast per §F, targets ≥ 48dp, TalkBack labels on every state change, reduced-motion fallbacks per W5.
