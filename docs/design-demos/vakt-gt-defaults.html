VAKT GT — the two black-hole registers, fixed

On a fresh install the top register and the big lower register showed nothing — a dark hole inside the ring (your photo). The machined instrument art only ever appears if you tap the slot and pick “Empty”, which nobody does. This preview shows the fix: give each register a sensible default so it fills with real data the moment the watch face goes on. Use the pickers to choose what each one shows.
